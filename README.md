# Анализ динамики продаж и влияния погодных условий
### Этот проект посвящен исследованию данных о продажах и их корреляции с температурой воздуха в Астане. В ходе работы были применены методы статистического анализа и визуализации данных на языке Python.
## Основные этапы работы:
#### Предобработка данных: Очистка датасета от выбросов с использованием метода IQR (межквартильный размах).
#### Анализ временных рядов: Визуализация динамики продаж и выявление сезонных колебаний.
#### Интеграция внешних данных: Парсинг и обработка метеорологических данных (температуры) для сопоставления с бизнес-показателями.
#### Интеграция внешних данных: Парсинг и обработка метеорологических данных (температуры) для сопоставления с бизнес-показателями.
#### Таргетированная фильтрация: Поиск закономерностей продаж конкретных товаров в зависимости от дня недели и склада.
## Технологический стек:
#### Pandas — манипуляция данными и агрегация.
#### Matplotlib — построение графиков.
#### NumPy — математические вычисления
